# Linux Permissions

## Viewing permissions 
- `ls -l`
Shows the permissions for files.
- `ls -la`
Shows permissions and hidden files.
- `ls -ld` 
Shows the permissions of the directory itself.

- ` -`
=file 
- `d` 
= directory

- `rwx` 
can read,write and execute 
- `r-x`
Can read and write but not execute
- ` r--`
Can only read 

---
## Permission types
- Read (r)
- Write (w)
- Execute (x)

--- 
## Who gets permissions?
- Owner
- Group 
- Others

## Change Permissions
` chmod +x filename`
Adds execute permission.

`chmod -x filename`
Removes execute permission.
