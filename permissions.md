# Linux Permissions

## Viewing permissions 
- `ls -l`
Shows the permissions for files and directiories.
- `ls -la`
Shows permissions and hidden files.
- `ls -ld` 
Shows the permissions folder

- ` -`
=file 
- `d` 
= directory

- `rwx` 
Owner
- `r-x`
Group
- ` r--`
Everyone else 

---
## Permission types
- Read (r)
- Write (w)
- Execute (x)

--- 
## Who gets permissions?
- Owner
